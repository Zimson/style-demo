# Demo for styled-jsx

#### Started

1. install dependencies
```bash
$ npm install
```

2. develop start: 
```bash
$ npm start
```

3. build test and prod
```bash
$ npm run build:all
```

4. build test/prod separately:
```bash
$ npm run build:prod

$ npm run build:test
```

#### Description

`outer-library` - external assembled component

`src` - application sources. sass, the pug is used. assembly with the gulp, webpack-stream.
